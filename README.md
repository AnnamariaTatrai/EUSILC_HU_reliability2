# EUSILC_HU_reliability2

This repository contains the code used in research on the quality of EU-SILC income data. It includes four R Shiny applications, as well as analyses examining gross–net income inconsistencies and the share of observations clustered around the poverty threshold.

Our work on this topic has been covered in the Hungarian press, with articles available in both Hungarian and English. In addition, journalist Anita Élő explored the issue from an investigative journalism perspective.
